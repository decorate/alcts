# Changelog

## [2.0.0] - 2024-03-06

### Changed

- `relation`を`Relation`ユーティリティクラスに移行
  - より直感的なインターフェース
  - 型安全性の向上
  - 循環参照の問題を解決

### Added

- より詳細なドキュメント

### Breaking Changes

- リレーションの実装が`Relation`クラスに変更
- リレーションの取得方法が変更（`relation` → `relation.get()`）

## [1.6.0] - 2024-03-05

### Added

- キーの自動変換機能（スネークケース ⇔ キャメルケース）
- より柔軟なデータ変換オプション

### Changed

- データ変換の処理を改善
- エラーハンドリングの強化
